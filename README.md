# CSE Fundamental

A dedicated repository for mastering the core pillars of Computer Science and Engineering. This project serves as a structured archive of algorithmic problem-solving, engineering mathematics, and full-stack application logic.

## 🏛️ Repository Structure

The contents are organized into four primary domains:

### 1. Algorithmic Problem Solving (C Language)
Focused on building a strong mental model for logic and memory management.
* **Mathematical Logic:** Implementations for GCD, Fibonacci sequences, and integer manipulations.
* **Pattern Geometry:** Coordinate-based rendering (e.g., fashionable "X" mark generation) using nested loops.
* **Control Flow:** Mastery of conditional logic and iterative processes.

### 2. Engineering Mathematics
Code implementations of academic concepts used in technical computing:
* **Linear Algebra:** Matrix operations including addition, multiplication, and transposition.
* **Number Theory:** Prime factorization and modular arithmetic.


### 3. Technical Physics & Electronics
Foundational simulations and troubleshooting documentation:
* **Simple Harmonic Oscillators:** Algorithmic representation of SHM.
* **Power Systems:** Troubleshooting logic for 12V power input systems and UPS hardware.

## 🚀 Getting Started

### Compilation (C Programs)
To run the algorithmic solutions, use any standard C compiler:
```bash
gcc -o solution main.c
./solution
```

### Database Setup
1. Import the provided `.sql` files into your local MySQL instance.
2. Update the `config.php` file in the dashboard directory with your local credentials (`127.0.0.1`).
